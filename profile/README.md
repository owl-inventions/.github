# Overview

This organization consolidates side projects and leverages GitHub's organizational features, including shared action runners and secrets—to streamline cloud infrastructure management. The infrastructure runs on a cost-effective K3S cluster with dynamic resource allocation via Hetzner HCLOUD.

**Maintainer:** [Jan Hasselberg](https://jnhssg.de)

## Projects

**Infrastructure & Tooling**
- Kubernetes-Terraform for K3S deployment on Hetzner Cloud
- Reusable GitHub Actions for CI/CD integration with K3S, AWS S3, and CloudFront
- Keycloak Terraform configuration for identity provider services
